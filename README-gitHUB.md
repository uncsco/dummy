## GitHUB - PAT

### Generate a PAT

Generate a PAT (Personal Access Token )
https://docs.github.com/en/authentication/keeping-your-account-and-data-secure/creating-a-personal-access-token

```
Settings
 - Developer settings
    - Personal access tokens
```

### Use PAT as password

Use the PAT as a password:

- e.g. when `push`-ing

```sh
git push

# // Type the username
# // Type the PAT
```
